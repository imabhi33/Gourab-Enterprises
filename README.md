# Gourab Enterprises

This repository contains the website frontend inside the `frontend/` folder.

## Deploy on Vercel

1. Import this repository into Vercel.
2. Set the project root directory to `frontend`.
3. Framework preset: `Vite`.
4. Add the environment variables from `frontend/.env.example`.
5. Deploy.

## Local Development

```powershell
cd frontend
npm install
npm run dev
```
